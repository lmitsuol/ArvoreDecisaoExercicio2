# ArvoreDecisaoExercicio2
Outro exercício com um tema mais complicado

Tema: Previsão de Aprovação de Crédito Bancário

O objetivo é prever se um cliente terá crédito Aprovado ou Negado com base em:

Renda (Alta/Média/Baixa)
Histórico de Crédito (Bom/Ruim)
Dívidas Atuais (Sim/Não)
Tempo de Emprego (Longo/Curto)
